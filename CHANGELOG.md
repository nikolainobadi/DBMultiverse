# Changelog

All notable changes to DBMultiverse will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [1.2.0] - 2025-12-22

### Added
- SwiftData versioned schema support for improved data model migration

### Changed
- Updated Swift package dependencies to stable versions

### Fixed
- Image parsing issue for balloon container images

## [1.1.0] - 2025-09-08

### Added
- Swift 6 language support with strict concurrency checking

### Changed
- App name updated to "Multiverse Reader"
- Updated app icon

## [1.0.0] - 2025-01-10

### Added
- Initial release of DBMultiverse iOS app
- Comic reading functionality with chapter navigation
- Chapter caching for offline reading
- SwiftData integration for tracking read chapters and progress
- Multi-language support for accessing different language versions
- Special universe sections for exploring chapters by universe
- iPad and iPhone platform support
- Direct web links to DragonBall Multiverse webcomic pages
